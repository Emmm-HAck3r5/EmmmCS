\chapter{Kernel Intr}
中断处理函数。

\section{intr\_init}
\texttt{void intr\_init(void)}

简介：对中断模块进行初始化。

\subsection{函数功能}

开机时被调用，初始化中断模块。

\section{intr\_open}
\texttt{void intr\_open(void)}

简介：打开中断。

\section{intr\_close}
\texttt{void intr\_close(void)}

简介：关闭中断。

\section{intr}
\texttt{void intr(void)}

简介：中断处理主函数，用于分发中断。

\subsection{函数功能}

在发生中断时被调用，读取 \texttt{0x7fffc} 开始的 4 子节作为中断参数，根据首字节进行分发；具体定义见 cpu 中断部分。
