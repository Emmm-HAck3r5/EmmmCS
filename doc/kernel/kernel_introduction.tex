\chapter{Kernel 综述}
\section{架构}
本Kernel不提供进程、线程模型，文件系统，网络。\\
本Kernel由以下部分组成：
\begin{itemize}
    \item KLIB - Kernel编程的基础库
    \item driver - 提供对VGA、PS/2 KB、LED等设备控制
    \item mm - 内存管理器，提供动态内存管理
    \item sh - 提供交互式命令行
    \item app - 内置应用
\end{itemize}
